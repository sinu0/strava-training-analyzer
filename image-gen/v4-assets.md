# V4 editorial cycling assets

The V4 hero assets are original images generated with the built-in image generator on 2026-07-19, then resized and compressed to WebP for the frontend.

| Asset | Intended screen | Prompt direction |
| --- | --- | --- |
| `hero-today-road.webp` | Dzisiaj | Solo road cyclist, sunrise mountain climb, copy-safe left side |
| `hero-analytics-cockpit.webp` | Analiza | Road-bike cockpit, blue-hour mountain road, no readable computer data |
| `hero-training-prep.webp` | Plan | Top-down kit, route map, bottles and preparation ritual |
| `hero-routes-climb.webp` | Trasy | Alpine switchbacks and a lone cyclist for route-planning context |

Every prompt excludes branding, readable text, watermarks and UI elements. Source PNGs are intentionally not kept in the repository; the WebP files are the production deliverables.
